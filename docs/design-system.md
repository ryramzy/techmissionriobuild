# TechMissionRio Design System

## Colors
- Primary: #0B1F3A (tmr-blue)
- Accent: Green → Blue gradient (tmr-gradient)
- Secondary: #22C55E (tmr-green)
- Tertiary: #14B8A6 (tmr-teal)
- Warm: #FB923C (tmr-orange)
- Light: #FCD34D (tmr-yellow)

## Principles
- Clean
- Trustworthy
- Youthful but professional
- Accessible

## Icon Usage
- Use full icon only in hero + PWA
- Use simplified mark for navbar
- Use watermark at 5–10% opacity for backgrounds

## Typography
- Primary: Syne (headings, branding)
- Secondary: Inter (body text, UI)

## Animations
- tmr-pulse-glow: Subtle pulsing glow effect
- tmr-float: Gentle floating animation
- tmr-gradient-shift: Animated gradient movement

## Components
- Navigation: Glassmorphism with gradient accents
- Cards: Subtle borders with hover states
- Buttons: Gradient backgrounds with glow effects
- Hero: Watermark + circuit patterns + gradient arcs

## Accessibility
- Maintain WCAG AA contrast ratios
- Use semantic HTML
- Provide alt text for all icons
- Ensure keyboard navigation
