# Branding Guidelines

## Voice
- Empowering
- Clear
- Global

## Visual Identity
- Rio + Technology fusion
- Minimalist with strategic gradients
- Circuit patterns representing connectivity
- Christ silhouette representing faith and location

## Logo Elements
- TMR Monogram: Technology focus
- Rio Silhouette: Christ + mountains
- Gradient Arc: Sunrise/optimism
- Circuit Pattern: Technology + future

## Do Not
- Distort icon
- Use low-contrast overlays
- Overcrowd layouts
- Change gradient directions
- Use full logo at small sizes

## Color Hierarchy
1. Primary: tmr-blue (#0B1F3A)
2. Accent: tmr-green (#22C55E)
3. Supporting: tmr-gradient
4. Text: tmr-white (#FFFFFF)

## Usage Guidelines
- Hero sections: Full logo with watermark
- Navigation: Simplified TMR mark
- Cards: Gradient accents only
- Backgrounds: 5-10% opacity watermarks
